pygcgen

A python version of GitHub Changelog Generator

Download .zip Download .tar.gz View on GitHub

Status

Currently i'm only testing on Win10 (64Bit) with Python 2.7 (3Bit).

Installation

python setup.py install

Usage

In your repository dir: pygcgen or python -m pygcgen. From elsewhere: pygcgen -u skywinder -p ActionSheetPicker-3.0 or python -m pygcgen -u skywinder -p ActionSheetPicker-3.0.
